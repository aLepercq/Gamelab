## Installation
* Récuperer le code depuis github.
* Ajouter le fichier .env au répertoire
* Lancer le server :
  * Ouvrir un terminal sur VScode / IntelliJ
  * se mettre à la racine du projet
  * npm start
* Lancer le client :
  * Ouvrir un terminal sur VScode / IntelliJ
  * /cd client
  * npm start
* Aller sur http://localhost:3000
